The text_indentation module
=======================


Module to import to use ``text_indentation``

    >>> text_indentation = __import__("5-text_indentation").text_indentation

Usage

    >>> text_indentation("Hello?")
    Hello?
    <BLANKLINE>

    >>> text_indentation("Houston? It's Buzz.")
    Houston?
    <BLANKLINE>
    It's Buzz.
    <BLANKLINE>

    >>> text_indentation("Do you copy? It's alive. I repeat: alive.")
    Do you copy?
    <BLANKLINE>
    It's alive.
    <BLANKLINE>
    I repeat:
    <BLANKLINE>
    alive.
    <BLANKLINE>

    >>> text_indentation("?????")
    ?
    <BLANKLINE>
    ?
    <BLANKLINE>
    ?
    <BLANKLINE>
    ?
    <BLANKLINE>
    ?
    <BLANKLINE>

    >>> text_indentation("Help.Iam:Losing.Grrrr?")
    Help.
    <BLANKLINE>
    Iam:
    <BLANKLINE>
    Losing.
    <BLANKLINE>
    Grrrr?
    <BLANKLINE>

    >>> text_indentation(str(30 + 12) + "?" + str(845 - 179))
    42?
    <BLANKLINE>
    666

    >>> text_indentation(str(-8 - 10))
    -18
